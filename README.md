# ☁️ SKY-Y ✈️ 

> - BootStrap + jQuery Practice Project
> - Bootstrap을 이용한 responsive web 제작.
> - Bootstrap의 다양한 커스텀 클래스를 이용함.
> - jQuery의 `toggleClass`를 이용하여 스크롤 시 navBar의 배경색이 나타나도록 함.

<br>

## [🌐 배포 링크](https://thisisyjin.github.io/SKY-Y/)
- gh-pages로 배포함.
